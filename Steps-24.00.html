Adding Arrow Indicator 24

Arrow Max Salary %


Formula: (The indicator is hidden initially until a visual is selected)
Arrow Max Salary % = 
VAR Change = 
    DIVIDE(
        AVERAGE('Glassdoor_Salary_Cleaned'[max_salary]) - [National Avg Max Salary],
        [National Avg Max Salary],
        0
    )
VAR RowCheck = COUNTROWS(VALUES('Glassdoor_Salary_Cleaned'[Job Title]))  -- Change if needed
RETURN
SWITCH(
    TRUE(),
    RowCheck = 0, " ",               // No row context — hide
    ISBLANK(Change), " ",            // Blank value — hide
    Change = 0, "",                  // No change — hide
    Change < 0, UNICHAR(9660),       // ? Down
    Change > 0, UNICHAR(9650)        // ? Up
)


 Click on canvas, click Card visualizations, drag Arrow Max Salary % to fields value box

visual Format
a.  In the format-> Formatting Tab-->Visual-->Callout value: Font = Arial, size =15,
 color -> click fx (conditional formatting) set format style= rules, field based on = _Max Salary %
            If value >=0 and <=max then #03983A
            If value >=min and <0 then #BC0104,
        Category label= Off
    b. In the format->General-->Properties-->Size: Height =32, Width = 44, Position: Horizontal = 156, Vertical = 220, Effect: Background: On, Color = White, Transparency = 100%.


Follow step by step instruction.
