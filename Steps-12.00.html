Analyze and Clean Data 12
In this step, you will clean and transform the data:
  1. Select Transform Data from the Home tab.
  2. Check if the dataset has any inconsistencies, such as:
    • Inconsistent formats (e.g., date formats like MM-DD-YYYY and YYYY/MM/DD).
    • Null values or missing data in critical fields.
    • Duplicate records or other data quality issues.
    • The data need transformation. 
  3. If no inconsistencies are found, confirm that the data is clean and consistent. 
  4. Review the data.
    • To ensure accuracy across all columns after transformation and make sure all Data Types are formatted correctly. 
  5. Click on File and select Close and Apply to load the data into Power BI.
Look at the below step-by-step.

Removed Duplicates
Go to your query editor in your Power BI top ribbon and by clicking on transformation you will get to the query editor. Here is where you identify the issues and resolve. By finding the correct column, you click on the column then right click and you will scroll down and click "Remove Duplicates"




Removing null values from all Columns
In your query editor you want to replace the (-1) as a null, then filter each column that needs transformation by deselecting the null, then in the ribbon in the manage columns, you want to click on remove blanks.
