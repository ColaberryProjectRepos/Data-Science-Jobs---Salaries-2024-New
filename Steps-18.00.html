Creating All Measures Table 18


1.  In Home Tab, select Enter data and Create New table
2.  Rename the table to ‘All Measures’ Table
3.  Click Load
4.  Right Click on 'All Measures' and select New Measure
5.  Copy and paste the DAX formula Measure for "Average Salary" in the DAX bar.
6.  Click the Green Check to create measure. 
7.  Repeat step 4-6 until you create all needed measures.

Follow step by step instruction.




DAX Measure Formula:
1. Total Number of Jobs = COUNT('Glassdoor_Salary_Cleaned'[Job Title])

2. National Avg Salary = CALCULATE( AVERAGE('Glassdoor_Salary_Cleaned'[avg_salary]), ALL('Glassdoor_Salary_Cleaned')
) 

3. National Avg Min Salary = CALCULATE(
AVERAGE('Glassdoor_Salary_Cleaned'[min_salary]), ALL('Glassdoor_Salary_Cleaned')

4. National Avg Max Salary = CALCULATE(
AVERAGE('Glassdoor_Salary_Cleaned'[max_salary]),
