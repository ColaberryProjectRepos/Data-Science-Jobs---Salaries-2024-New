Adding KPI's and Refresher Date Part 1 21




Total Number of Jobs KPI


Formula
Total Number of Jobs = COUNT('Glassdoor_Salary_Cleaned'[Job Title])

1. Click on an empty space on the canvas, and from the Build pane select Card   visual

2. On the build pane drag and drop the Total Number of Jobs measure to visual card.

Visual Format
In the Formatting Tab-->Visual-->Callout value: Font = DIN, size = 24, Color: #0D6ABF, Category label: Off
In the Formatting Tab-->General-->Properties-->Size: Height = 90, Width = 205, Position: Horizontal = 0, Vertical = 92,
Title: Jobs, Font = DIN, Size = 14, Color = #F1F1F1, Background Color= #094780, Center,
Effect: Background: On, Color = #F1F1F1, Transparency = 0
