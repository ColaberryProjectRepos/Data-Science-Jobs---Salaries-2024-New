Adding Calculated Column 20


1.  In Home Tab, select Modeling 
2.  Click Create New Column
3.  Put the formula for Calculated Column "_Category" in the Dax Bar

Formula:
_Category = 
SWITCH(
    TRUE(),
    SEARCH("Data Scientist", 'Glassdoor_Salary_Cleaned'[Job Title], 1, 0) > 0, "Data Science",
    SEARCH("Machine Learning", 'Glassdoor_Salary_Cleaned'[Job Title], 1, 0) > 0, "Data Science",
    SEARCH("AI", 'Glassdoor_Salary_Cleaned'[Job Title], 1, 0) > 0, "Data Science",
    SEARCH("Data Analyst", 'Glassdoor_Salary_Cleaned'[Job Title], 1, 0) > 0, "Data Analysis",
    SEARCH("Business Intelligence", 'Glassdoor_Salary_Cleaned'[Job Title], 1, 0) > 0, "Data Analysis",
    SEARCH("Data Engineering", 'Glassdoor_Salary_Cleaned'[Job Title], 1, 0) > 0, "Data Engineering",
    SEARCH("Big Data", 'Glassdoor_Salary_Cleaned'[Job Title], 1, 0) > 0, "Data Engineering",
    SEARCH("Research Scientist", 'Glassdoor_Salary_Cleaned'[Job Title], 1, 0) > 0, "Research & Science",
    SEARCH("Scientist", 'Glassdoor_Salary_Cleaned'[Job Title], 1, 0) > 0, "Research & Science",
    SEARCH("Consultant", 'Glassdoor_Salary_Cleaned'[Job Title], 1, 0) > 0, "Consulting & Specialist Roles",
    SEARCH("Director", 'Glassdoor_Salary_Cleaned'[Job Title], 1, 0) > 0, "Leadership & Management",
    SEARCH("Manager", 'Glassdoor_Salary_Cleaned'[Job Title], 1, 0) > 0, "Leadership & Management",
    SEARCH("VP", 'Glassdoor_Salary_Cleaned'[Job Title], 1, 0) > 0, "Leadership & Management",
    SEARCH("Principal", 'Glassdoor_Salary_Cleaned'[Job Title], 1, 0) > 0, "Leadership & Management",
    "Contract"
)
