Methodology 3

Data Preparation
Data was cleaned by identifying and removing duplicates, handling missing values, filtering outliers, and standardizing the data to ensure consistency and reliability, supporting the salaries based on their position, location, and prospective employer.




Model Selection
The Regression model, which is designed to predict continuous outcomes such as salary. Evaluation metrics reveal exceptional performance, with an R² value of 1, meaning the model explains 100% of the variance in the target variable. The error metrics are very low: Mean Squared Error (MSE) is 0.1, Mean Absolute Error (MAE) is 0.2, and Median Absolute Error is also 0.2, all indicating minimal prediction errors. A variance of 1 confirms the target values are evenly distributed, matching the model’s output well. The regression prediction graph further validates this, showing blue predicted points tightly aligned along the diagonal ideal-fit line. The intercept is 0.0 and slope is 1.0, reflecting a perfect 1:1 relationship between actual and predicted values. These coefficients mean that for every unit increase in actual values, the predicted values increase equally. Red dots mark exact alignment between predicted and actual values. Altogether, the visualizations and metrics confirm that the regression model is nearly perfect in prediction accuracy.




Model Development:
The model was trained on Data Science Jobs & Salary data. The training set enabled the model to identify key patterns, while the testing set ensured reliable predictions on unseen data, maintaining accuracy and generalization.
