Adding Sub KPI Cards 23


Max Salary Sub KPI

Formula: (The sub kpi is hidden initially until a visual is selected)
_Max Salary % = 
VAR Change = 
    DIVIDE(
        AVERAGE('Glassdoor_Salary_Cleaned'[max_salary]) - [National Avg Max Salary],
        [National Avg Max Salary],
        0
    )
RETURN
IF(
    Change = 0,
    "",  -- Hide if no change
    Change
)

Click on canvas, click card visualizations, drag _Max Salary % to fields value box, 

visual Format
a.  In the format-> Formatting Tab-->Visual-->Callout value: Font = DIN, size =15, color #252423, Category label= Off
     b.  In the format->General-->Properties-->Size: Height =32, Width = 55, Position: Horizontal = 150, Vertical = 239, 
Title = Off
Effect: Background: On, Color = White, Transparency= 100%.
