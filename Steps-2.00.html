Data Overview 2

Data Sources and KPIs:
The Data Science Jobs & Salary dataset of 2024 is a salary dynamic across various job roles, industries, and geographic regions that was gathered through Microsoft SQL Server. This dataset is available on Kaggle, a reputable online data repository. It is structured to predict salary outcomes based on key input variables such as job title, location, and company attributes.





System Architecture
The architecture focuses on analyzing salary across job roles, industries, and geographic regions. It involves gathering historical sales data, preprocessing and cleansing it, selecting key performance metrics, and applying data modeling techniques using Python. Predictive models are built using Spark for scalable data processing, and insights are visualized in Power BI to drive strategic decision-making. The project integrates Microsoft Fabric with Power BI for data visualization and reporting, enabling enhanced decision-making and operational efficiency across all sales channels.


Data Features
The dataset includes key variables such as Max Salary, Min Salary, Avg Salary, High Salary Job%, Job Rating. These features are essential for evaluating salary ranges by Job roles and Geographical area. The data supports decision-making processes by expected salaries based on their position, location, and prospective employer, while HR teams strive to structure competitive compensation packages.
